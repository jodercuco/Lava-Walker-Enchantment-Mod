# 🔥 Lava Walker Enchantment Mod

![Lava Walker Icon](https://raw.githubusercontent.com/jodercuco/Lava-Walker-Enchantment-Mod/main/src/main/resources/assets/lava-walker-enchantment-mod/icon.png)

A lightweight and vanilla-friendly Fabric mod that introduces the **Lava Walker** enchantment. Traverse the dangerous lakes of the Nether with ease by turning lava into solid rock beneath your feet.

---

## 🛠️ Features

* **Lava Transformation:** Walk over lava source blocks to instantly turn them into **Molten Basalt**.
* **Dynamic Block States:** The created basalt isn't permanent. it has **3 visual stages** of cooling and cracking before eventually melting back into lava.
* **Temperature Mechanics:** Molten Basalt is unstable! It melts significantly **faster in the Nether** due to the extreme heat, while it remains stable for longer in the Overworld.
* **Two Levels:** * **Level I:** Small radius for careful crossing.
    * **Level II:** Wider radius, perfect for sprinting across large lakes.

## ❄️ Compatibility & Balance

* **Incompatible with Frost Walker:** To keep the game balanced, you cannot have both Frost Walker and Lava Walker on the same pair of boots.
* **Requirements:** Only works on **Lava Source blocks** with **Air** above them. It won't work on flowing lava or underwater (underlava) caves.

## 💎 How to Obtain

You can find the Lava Walker enchantment through:
* **Enchantment Table:** Available as a standard enchantment for boots.
* **Trading:** Can be found in Librarian Villager trades.
* **Loot:** Occasionally found in Bastion Remnants and Ruined Portals.

---

## 🚀 Installation

1.  Make sure you have **Fabric Loader** installed.
2.  Download the latest version from [Modrinth](https://modrinth.com/mod/lava-walker-enchantment-mod).
3.  Place the `.jar` file and the **Fabric API** in your `/mods` folder.
4.  Enjoy walking on fire!
